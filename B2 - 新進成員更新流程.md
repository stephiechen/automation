# B2 | 新進成員更新流程

## 觸發來源
醫療處通知 → 新進成員（區分：醫師 / 助理）

---

## 執行步驟

```
醫療處通知新進成員（標明：醫師 or 助理、所屬院所）
    ↓
取得照片 + 學經歷 / 職稱資料
    ↓
委製介紹圖（3 種尺寸）
    ↓
InfoSync 後台：填資料 + 上傳照片 + 選擇類別 + 選擇院所
    ↓
系統依 order 規則插入正確位置
    ↓
驗收確認
```

---

## 排序規則

同院所的醫師與助理排在一起，院所依地理位置排列：

```
院所 A（依地理位置）
  └ 醫師（職等排序 TBD）
  └ 助理

院所 B
  └ 醫師
  └ 助理

院所 C ...
```

**Shopline order 欄位操作：**
- 可使用小數點插入（例如在 3 和 4 之間插入用 3.5）
- 助理從序號要新增在該院別醫師後面

---

## InfoSync 後台操作說明

開啟 官網後台 → 切換至「**醫師資訊**」Tab

| 欄位 | 填寫內容 |
|------|---------|
| 姓名 | 顯示於官網卡片 |
| 院所 | 選擇所屬院所 |
| 類別 | 醫師 / 助理 |
| 科別 / 職稱 | 醫師填科別，助理填職稱 |
| 學經歷 / 簡介 | 依照提供資訊，依照欄位填入 |
| 照片 | 上傳 Canva 調整好比例的圖 |

---

## 驗收確認項目

- [ ] 官網醫師陣容頁：新成員出現在正確院所區塊
- [ ] 醫師排在助理前面
- [ ] 圖片比例正確（Canva 已調整）
- [ ] 廣告機素材更新（若適用）

---

## 注意事項

- 圖片建議先用 Canva 調整比例再上傳，避免官網顯示變形
- 助理序號從 53 起，與醫師序號區分
- 職等排序規則 TBD，確認後更新此文件
